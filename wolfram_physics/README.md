# Wolfram Physics

This folder collects an OPH and Wolfram line of work on observer-semantic pruning of the Ruliad.

Using Senchal's Observer Theory and OPH's observer overlap consistency, this line of work narrows rule space to rules that generate universes with observers that persist and make compatible observations. The research target is the "God Rule": a rule whose observer-visible universes survive these filters and settle into a stable observer-semantic structure.

The most technical description in this folder is [Finite Survivor Quotients and Bridge Dynamics for Observer-Semantic Ruliad Pruning](papers/finite_survivor_quotients_and_bridge_dynamics.pdf). Senchal's public reference paper is [Observer Theory and the Ruliad: An Extension to the Wolfram Model](https://github.com/SASenchal/Observer-Theory-Extension/blob/main/Observer_Theory_and_the_Ruliad_An_Extens.pdf).

The toy benchmark gives a concrete pruning result. After the confluence and holonomy filters, 92 of 154 raw rule families survive, so only 59.7% survive and 40.3% are excluded. At the semantic-class level, 2 of 8 classes survive.

The folder also includes the benchmark code, the survivor-bridge code, and the figure assets used by the papers.
